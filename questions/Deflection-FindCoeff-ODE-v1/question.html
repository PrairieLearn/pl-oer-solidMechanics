The simply supported beam has length $L$, elasticity modulus $E$, and cross-section with moment of inertia $I$. A concentrated force is applied at half point, as illustrated below

The deflection curve for the the first half of the beam is given by:

$ y_1(x) = \frac{P}{EI} \left( - \frac{L^2 x}{16} + \frac{x^3}{12} \right)$

Obtain the equation for the deflection curve $y_2(x)$ for $L/2 \lt x \lt L$, where:

$ y_2(x) = \frac{P}{E I} \left(A_o + A_1 \ x + A_2 \ x^2 + A_3 \ x^3 \right )$

When solving this problem symbolically, you will notice that the coefficients above depend only on the length of the beam $L$. Determine the remaining coefficients when $L = {{params.L}} \rm\ m$.

$ y_2(x) = \frac{P}{E I} ($ + $x$ + $x^2$ + ${{params.a3_2}} \ x^3 ) \rm\ [m]$
